The question

Which face carries the wordmark?

Lilita One is the current canon. The brand bible justifies it as "showman / candy-bar / theme-park signage — maps directly to Winston the Earth-side importer-showman." Bowlby One is the alternative on the table — closer to pulp-comic and importer-poster register. Bowlby One SC is the small-caps sister; included for completeness but only meaningful for mixed-case input. Same canon treatment applied to all three: orange fill · ink + lime stacked shadow · skewX(-6deg).

My read

Honest recommendation

Drop Bowlby One SC immediately.

The "SC" in Bowlby One SC means small-caps. It only changes how lowercase letters render (as small caps with reduced cap height). The Wittyverse wordmark canon is ALL CAPS — every letter is uppercase input. Bowlby One SC and Bowlby One render visually identically for "WITTYVERSE". The SC variant only earns its keep if you ever set the brand name in mixed case (which canon NEVER 15 forbids).

The real choice is Lilita One vs Bowlby One. Two honest cases:

Case for Lilita (status quo): The bible's "candy-bar / theme-park signage" framing is doing real work. Lilita's wide friendliness is the differentiator — it makes Winston the playful importer-showman, not the aggressive one. The Onion / Liquid Death / Meow Wolf references in the bible all lean toward "premium craft applied to absurd retail" — friendly, not threatening. Lilita supports that read. Bowlby's pulp-tabloid edge is closer to "aggressive late-night-TV pitchman" than "kindly bazaar owner."

Case for Bowlby: The brand's visual register is loud, condensed, polychrome — that's the v0.7 work. Bowlby matches that register better than Lilita does. Lilita is the soft note in an otherwise hard-edged brand. Bowlby unifies the wordmark with the display copy as one poster family. The pairing test above shows this clearly — Bowlby + BSD read as siblings; Lilita + BSD read as a label glued on a press release. If you want maximum visual cohesion across the brand, Bowlby wins.

What I'd actually do

Keep Lilita One as canon for the wordmark, but bring Bowlby One into the system as the secondary display face for poster moments — homepage hero banners, marquee strips, social ad headlines, retail signage. That gives you Lilita's friendliness for the brand-mark itself (where the showman-not-thug read matters), AND Bowlby's pulp punch for the loud display moments that need to land harder than BSD alone provides.

Concretely: add --font-pulp: 'Bowlby One', sans-serif as a fifth typeface. Reserve it for hero banners and marquee display (one use per page max). Existing BSD stays as the workhorse for everything else.

If you want to test Bowlby as the wordmark anyway, the natural test is to update the topbar masthead on the brand-guidelines page itself and see if it feels right there for a few days before committing.
